# RelayFun - リレーマラソン管理・応援アプリ

## 概要
Fukuoka Relay Marathon（42.195km）をチームで完走するためのタイム管理およびコミュニケーション支援Webアプリケーションです。
目標タイム（4時間）に対するペース管理機能に加え、チームごとのユニークな世界観（「太田ヨットスクール」、「徳永敦アゲ♂Surfing☆スクール」など）を反映したエンターテインメント性の高いUIが特徴です。

## 特徴
- **チーム別テーマデザイン**:
  - **太田ヨットスクール**: 爽やかな青と白を基調としたマリン/ヨットテーマ。
  - **徳永敦アゲ♂Surfing☆スクール**: 派手なグラデーションとギャル語満載のサーフィン/ギャルテーマ。
- **レース管理機能**:
  - 1周約1.4km × 30周のラップタイム計測。
  - 4時間完走ペース（キロ5分42秒）に対する貯金/借金のリアルタイム表示。
  - 走順のカスタマイズとドラッグ＆ドロップによる入れ替え（実装状況による）。
  - レース中の「タスキ！」ボタンによる直感的なラップ計測。
- **データ永続化**:
  - LocalStorageを使用し、ブラウザを閉じてもレース状況を復元可能。
- **誤操作防止**:
  - リセット時のパスワードロック（`smed`）。
  - 直前のラップ操作取り消し機能（Undo）。

## 使用技術
- **Frontend**: React, Vite
- **Styling**: Tailwind CSS, clsx, tailwind-merge
- **Icons**: Lucide React
- **Deploy**: Vercel (想定)

## セットアップと起動方法

### 1. 依存関係のインストール
`relay-app` ディレクトリに移動し、パッケージをインストールします。

```bash
cd relay-app
npm install
```

### 2. 開発サーバーの起動
以下のコマンドでローカル開発サーバーを起動します。

```bash
npm run dev
```

ブラウザで `http://localhost:5173` （または表示されたURL）にアクセスしてアプリを確認してください。

### 3. ビルド
本番環境用にビルドする場合は以下のコマンドを実行します。

```bash
npm run build
```
