# まなびポケット デザインパターン

## カラーパレット

| 用途 | カラー | カラーコード |
|------|--------|-------------|
| サイドバー / メインカラー | ダークブルー / ネイビー | #2D4A6F |
| カードヘッダー / アクセント | 水色 / スカイブルー | #5BC0DE |
| CTAボタン | ターコイズ / ティール | #00B4B4 |
| 背景 | 薄いグレー | #F5F5F5 |
| カード背景 | 白 | #FFFFFF |
| アラート（要返信など） | オレンジ | #FF6B35 |

## UIコンポーネント

### カード
- 白背景
- 控えめなシャドウ
- 角丸（border-radius: 8px〜12px程度）

### ボタン
- プライマリ: ターコイズ/ティール系
- 角丸デザイン
- ホバー時に少し暗くなる

### サイドバー
- ダークブルー/ネイビー背景
- 白またはライトカラーのテキスト・アイコン

### レイアウト
- 余白を多めに確保
- カード型レイアウト
- 視認性重視

## 全体トーン

- クリーンで落ち着いた業務システム寄りのデザイン
- 教育向けで親しみやすい
- 児童・生徒・教職員・保護者が使いやすいユニバーサルなUI
- シンプルなアイコンと余白を活かしたレイアウト

## モック作成用プロンプト

UIモックを作成する際は、以下のプロンプトを使用してください：

> ネイビー×水色基調、白カード、ティールCTA、クリーンで落ち着いた教育システムUI

### バリエーション

- **シンプル版**: まなびポケット風：紺サイドバー、水色アクセント、白カード、業務システム調の清潔感
- **機能重視版**: 学校向けプラットフォームUI。ダークブルー×水色、カード型、余白多めで視認性重視

## 注意事項

- オレンジはメインカラーではなく、「要返信」などのアラート用途に限定して使用
- 派手な装飾は避け、落ち着いたトーンを維持
- アクセシビリティを考慮し、コントラスト比を確保
